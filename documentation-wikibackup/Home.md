# Welcome to the Rekindle Wiki

Rekindle is a self-hosted comic, manga, and book server. Run the server on any machine that holds your archive files, then connect from any device running the Rekindle client to browse your libraries, read natively, and download for offline use. Multiple users are supported with individual reading progress and permission levels.
